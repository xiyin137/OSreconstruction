## Chapter IV §2. Analytic continuation of the distributions

(Book p. 72-74; PDF p. 44.)

### Step 1: Spectrum-condition support property

Spectrum-condition Axiom III gives that the vector-valued distribution

```
Ã(p_1) Ã(p_2) ⋯ Ã(p_n) Ω = Φ̃_n(p_1,…,p_n)
```

is supported on the set where the partial sums
`q_k = p_k + p_{k+1} + ⋯ + p_n` lie in `supp dE = s_+` (the closed forward
mass-shell cone) for `k = 1,…,n`.

### Step 2: Admissible exponential test functions

Because `Φ̃_n` is supported on these "ordered" partial-sum cones, it admits
exponential test functions of the form

```
(3)  exp i[ q_1·z_1 + q_2·(z_2 - z_1) + q_3·(z_3 - z_2) + ⋯ + q_n·(z_n - z_{n-1}) ]
     = exp(i Σ_k p_k·z_k)
```

whenever

```
(4)  Im z_1 ∈ V_+,   Im(z_k - z_{k-1}) ∈ V_+ for k = 2,…,n.
```

These exponentials decay fast on `supp Φ̃_n` together with all derivatives, so

```
(5)  Φ_n(z_1,…,z_n) = ∫ exp(i Σ p_k·z_k) Φ̃_n(p_1,…,p_n) dp
```

is well-defined and **holomorphic** (vector-valued) in the absolute forward
tube `𝔖_{n+1}`.

### Step 3: Boundary value to the original distribution

Writing `z_k = x_k + i y_k`, `η_k = y_k - y_{k-1}` and reading `Φ_n(z)` as a
distribution in `x_1,…,x_n` parameterized by `η_1,…,η_n ∈ V_+`, one finds

```
(9)  lim_{η→0, η_k∈V_+}  Φ_n(x_1+iy_1, …, x_n+iy_n)  =  Φ_n(x_1,…,x_n)
```

in the sense of tempered distributions on `ℝ^{n(d+1)}`.

**Theorem (Jost IV §2).** The distribution `Φ_n(x_1,…,x_n)` is the boundary
value of the vector-valued analytic function `Φ_n(z_1,…,z_n)` regular in

```
(11)  Im z_1 ∈ V_+,   Im(z_k - z_{k-1}) ∈ V_+,   k = 2,…,n.
```

**Corollary.** Taking matrix elements with the vacuum Ω, we obtain that the
scalar Wightman distribution `𝔚_n(x_0,…,x_n) = 𝔚_{n+1}(x_0,x_1,…,x_n)` is the
boundary value of an analytic function `𝔚_n(z)` holomorphic on the absolute
forward tube `𝔖_{n+1}`.

**Corollary (difference-variable form).** Writing `ξ_k = x_k - x_{k-1}`,

```
W_n(ξ_1,…,ξ_n) = 𝔚_{n+1}(x_0,x_1,…,x_n)
```

is the boundary value of an analytic function `W_n(ζ_1,…,ζ_n)` regular on the
**forward tube in difference variables**:

```
𝔗^n = { (ζ_1,…,ζ_n) ∈ (ℂ^{1+d})^n : Im ζ_k ∈ V_+ }.
```

**Lean substrate.** `BHW.bvt_F OS lgc n` is exactly this difference-variable
analytic function `W_n`; `BHW.ForwardTube d n` is `𝔗^n`. Boundary value
recovery is captured by the moving-real-ray theorems in
`BoundaryValuesBase.lean`.

## Chapter IV §3. Complex Lorentz transformations

(Book p. 75-77; PDF p. 44-45.)

### Definitions

A complex Lorentz transformation is a matrix `A ∈ ℂ^{(1+d)×(1+d)}` with

```
Aᵀ G A = G,   det A = ±1.
```

The kernel of `A ↦ det A` is the **complex Lorentz group** `L+(C)`. Unlike the
real restricted Lorentz group, `L+(C)` is *connected*.

```
(1)  L(C) = L+(C) + T·L+(C)    where T is the time reflection.
```

### Cayley parameterization (local chart at the identity)

A skew-symmetric matrix `R^T = -R` and a complex Lorentz transformation `A`
are related by

```
A = (I - GR)(I + GR)^{-1},   R = G(I - A)(I + A)^{-1}.
```

`R` is purely real ⇔ `A ∈ L↑_+`. So `L+(C)` is a complex extension of the real
restricted Lorentz group, with `L↑_+` as a totally real subgroup.

### Equivalence and normal forms

Two complex Lorentz transformations `A, B` are *equivalent relative to* `L↑_+`
if `A = Λ_1 B Λ_2` for some `Λ_1, Λ_2 ∈ L↑_+`. Each equivalence class has a
canonical **normal form** (Jost Theorem on p. 77, proved in Appendix I).

In odd dimension, normal forms of `A ∈ L+(C)` are:

(a1) `N = block-diag(L(iφ), K(iχ_1), …, K(iχ_k))`, `|φ| ≤ π`
(a2α) same with `L(iφ)` replaced by `1` (limit case)
(a2β) same with an extra reflection `R` in the off-block

In even dimension: normal-form blocks `K(iχ)` only, possibly with one `L(iφ)`
or `M(iτ)` block (exceptional case).

Here `K(φ)` is a spatial rotation, `L(χ)` is a boost (in two real time-space
directions), and `M(τ)` is the parabolic one-parameter subgroup (null
rotation). For complex parameters these become `K(iχ)` (analytic continuation
of rotation), `L(iφ)` (analytic continuation of boost), `M(iτ)` (analytic
continuation of null rotation).

**Lean substrate.**
- `L+(C)` ≅ `ComplexLieGroups.ComplexLorentzGroup d`
- Cayley chart: `ComplexLieGroups.Complexification` infrastructure
- Normal forms: `ComplexLieGroups.JostPoints` / `Complexification.lean`
- Connectivity: `ComplexLorentzGroup.isPathConnected` (proved)

## Chapter IV §4. The BHW theorem

(Book p. 78-80; PDF p. 46-47.)

**BHW-Theorem (Jost IV §4).** If the `L↑_+`-invariant analytic function
`W_n(ζ_1,…,ζ_n)` is regular and holomorphic in `𝔗^n` (forward tube in
difference variables), then it admits a single-valued, `L+(C)`-invariant
analytic continuation into the **extended tube**

```
𝔗'_n = ⋃_{A ∈ L+(C)} A·𝔗^n.
```

**Proof structure.**

Abbreviate `W_A(ζ) = W(A^{-1}ζ)` for `A ∈ L+(C)`.

1. `W_A` is well-defined on `A·𝔗^n`.

2. To extend `W` to `𝔗'_n` we must show `W_A(ζ) = W_B(ζ)` whenever
   `ζ ∈ A𝔗^n ∩ B𝔗^n ≠ ∅`. Otherwise the extension would be multi-valued.

3. Single-valuedness on the union `𝔗'_n` then implies the extended function is
   `L+(C)`-invariant.

4. The intersection `𝔗^n ∩ A𝔗^n` is an intersection of two open convex
   tubes — connected. So it suffices to prove `W = W_A` in a neighborhood of
   one point of `𝔗^n ∩ A𝔗^n`. If `W(ζ) = W_{A_0}(ζ)` for some `A_0 ∈ L+(C)`,
   the same is true in a neighborhood of `A_0`.

5. **Key step (germ of identity).** Fix `ζ ∈ 𝔗^n ∩ A_0 𝔗^n`. Use Cayley
   coordinates `A = A_0 · B`, with `B` in a connected neighborhood of the
   identity. `L↑_+`-invariance gives `W_A(ζ) = W_{A_0}(ζ)` for `B ∈ L↑_+`.
   `W_A(ζ)` is analytic in the Cayley parameters of `B`; it is constant on a
   totally real subset (`L↑_+`) and therefore — by the identity theorem
   `[BM 1*]` — constant on a complex neighborhood. So `W_A = W_{A_0}` in a
   neighborhood of `A_0`, and in particular at the identity `A_0`.

6. **Application of the identity theorem in the variable `A` (not `ζ`)** is
   essential: the totally real subset `L↑_+` provides the identity-theorem
   germ.

7. **Connectivity lemma.** The set
   `𝔅 = { A ∈ L+(C) : A·𝔗 ∩ 𝔗 ≠ ∅ }` is **connected**. Any point of `𝔅` can be
   joined to the identity by a finite chain of overlapping neighborhoods of
   the type used in steps 5-6.

8. **Proof of the connectivity lemma.** `𝔅` is a union of `L↑_+`-equivalence
   classes (by `Λ_{1,2} 𝔅 Λ_{1,2} = 𝔅`), so it suffices to check that each
   normal form lies in `𝔅`, and that normal forms continuously connect to the
   identity by decreasing parameters to zero. Both checks are explicit using
   §3 normal-form list.

This is the **BHW theorem**. The extension `W_n` to `𝔗'_n` is single-valued,
`L+(C)`-invariant, and analytic; `𝔗'_n` is a *domain of holomorphy* (it
contains real points — see §5).

**Lean substrate.**
- The extension itself: `BHW.extendF` defined in `ComplexLieGroups/BHWCore.lean`
- Analyticity: `extendF_holomorphicOn` in `JostPoints.lean`
- `L+(C)`-invariance: `complex_lorentz_invariance` in
  `ComplexLieGroups/Connectedness/ComplexInvariance/Extend.lean`
- Boundary value identification: `extendF_eq_on_forwardTube`

## Chapter IV §5. Application of BHW to locality — KEY

(Book p. 81-84; PDF p. 47-48.)

### Setup

`W_n(ζ_1,…,ζ_n)` extends analytically to `𝔗'_n` and is `L+(C)`-invariant.
Correspondingly,

```
(1)  𝔚_{n+1}(z_0, z_1, …, z_n) = W_n(z_1 - z_0, …, z_n - z_{n-1})
```

extends to the **absolute extended tube** `𝔖'_{n+1}` defined by

```
𝔖'_{n+1} = { (z_0,…,z_n) : (z_1 - z_0, …, z_n - z_{n-1}) ∈ 𝔗'_n }.
```

After extension, `𝔚_{n+1}` is invariant under the identity component of the
complex inhomogeneous Lorentz group, generated by complex translations and
`L+(C)`.

### Theorem (Jost III, on real points of 𝔗'_n)

```
Theorem [Jo 3].  The real point (ρ_1,…,ρ_n) is in 𝔗'_n if and only if
the convex cone 𝔱(ρ_1,…,ρ_n) generated by ρ_1,…,ρ_n contains only
space-like points.
```

**Proof.**

(a) Necessity. If `(ρ_1,…,ρ_n) ∈ 𝔗'_n`, there is `A ∈ L+(C)` with
`ρ_k = Aζ_k` and `ζ_k ∈ 𝔗`. Writing `ζ = Σ λ_k ζ_k` for any `λ_k ≥ 0`,
`ζ ∈ 𝔗` (convexity), so `(ζ, ζ) < 0`. Then
`(Σ λ_k ρ_k, Σ λ_k ρ_k) = (Aζ, Aζ) = (ζ, ζ) < 0`. So every nonzero element of
the cone is spacelike.

(b) Sufficiency. If `𝔱(ρ)` is purely spacelike, take a tangent plane `(α,ξ)=0`
between `V_+` and `𝔱`, and another `(β,ξ)=0` between `V_-` and `𝔱`. After
normalization `α = (1,1,0,…,0)`, `β = (-1,1,0,…,0)`. The conditions
`(α,ξ) > 0 for ξ ∈ V_+` and `(α,ξ) < 0 for ξ ∈ 𝔱` give `ξ^0 < -ξ^1` for
`ξ ∈ 𝔱`; similarly `ξ^0 < ξ^1`. Together: `ξ^1 > |ξ^0|`. The complex Lorentz
transformation

```
       ( 0  i  0  …  0 )
       ( i  0  0  …  0 )
(3)  A =( 0  0  1  …  0 ) ∈ L+(C)
       ( ⋮  ⋮  ⋮  ⋱  ⋮ )
       ( 0  0  0  …  1 )
```

then maps `ξ ∈ 𝔱` into `Aξ = ζ ∈ 𝔗`. Since `ρ_k ∈ 𝔱` we find
`A·ρ_k = ζ_k ∈ 𝔗`, i.e. `(ρ_1,…,ρ_n) ∈ A^{-1}·𝔗 ⊂ 𝔗'_n`.

### Corollary (Jost spacelike points)

The real difference variables `(ρ_1,…,ρ_n)` correspond to the real absolute
points `(r_0, r_1, …, r_n)` via `ρ_k = r_k - r_{k-1}`. Then

```
(4)  (r_k - r_l)² = - (ρ_{l+1} + ⋯ + ρ_k)² > 0
```

so all pairs `(r_k, r_l)` are *totally spacelike*. The real points of
`𝔖'_{n+1}` are exactly the totally spacelike configurations.

### Application of locality

Locality implies that on the real open set of totally spacelike points,
`𝔚_n(x_1,…,x_n)` is invariant under permutations of `(x_1,…,x_n)`:

```
(5)  𝔚_n(r_1,…,r_n) = 𝔚_n(r_{k_1}, r_{k_2}, …, r_{k_n}).
```

This equation analytically continues to an equation for Wightman functions

```
(6)  𝔚_n(z_1,…,z_n) = 𝔚_n(z_{k_1}, z_{k_2}, …, z_{k_n})
```

valid in the **smallest domain containing 𝔖'_n and invariant under all
permutations** applied to `(z_1,…,z_n)`. This new and extended domain is

```
𝔖^p_n = ⋃_π π·𝔖'_n
```

(union of all permuted extended tubes). Thus we have proved:

**Theorem (Jost IV §5, locality consequence).** The Wightman functions
`𝔚_n(z_1,…,z_n)` of a local field theory are regular analytic in the domain
`𝔖^p_n` and symmetric in `z_1,…,z_n`.

### Converse direction (Theorem on locality from symmetry)

```
Theorem.  If 𝔚_{n+1}(z_0,…,z_n) has all properties of a Wightman function
except possibly locality, and if in addition 𝔚_{n+1} is symmetric in
(z_0,…,z_n), then it satisfies the requirements of locality.
```

**Proof.** From the assumptions, `𝔚_{n+1}(z_0,…,z_n) = W_n(z_1-z_0,…,z_n-z_{n-1})`
with `W_n` holomorphic in `𝔗^n` and `L↑_+`-invariant. The BHW theorem extends
`W_n` to `𝔗'_n` and `𝔚_{n+1}` to `𝔖'_{n+1}`. The symmetry hypothesis extends
that to `𝔖^p_{n+1}`. We need to show the boundary distribution

```
𝔚_{n+1}(x_0,…,x_n) = W_n(x_1-x_0, …, x_n-x_{n-1})
```

satisfies

```
(7)  𝔚_{n+1}(x_0,…,x_{k-1},x_k,…,x_n) = 𝔚_{n+1}(x_0,…,x_k,x_{k-1},…,x_n)
     whenever (x_k - x_{k-1})² < 0.
```

**Explicit transposition argument.** Swapping `z_{k-1} ↔ z_k` corresponds to
the linear transformation on `(ζ_1,…,ζ_n) = (z_1-z_0,…,z_n-z_{n-1})`:

```
ζ_{k-1} → ζ_{k-1} + ζ_k
ζ_k     → -ζ_k
ζ_{k+1} → ζ_{k+1} + ζ_k
all other ζ_l fixed.
```

For `(ζ_1,…,ζ_n) ∈ 𝔗^n` (regularity domain of `W_n`):

```
(8)  W_n(ζ_1,…,ζ_{k-1}, ζ_k, ζ_{k+1},…,ζ_n)
   = W_n(ζ_1,…,ζ_{k-1}+ζ_k, -ζ_k, ζ_{k+1}+ζ_k,…,ζ_n).
```

Extension claim: this equation still holds if `ζ_l ∈ 𝔗` for all `l ≠ k`, but
`ζ_k = ξ_k` real with `(ξ_k, ξ_k) < 0`. Reason: under these conditions, the
point `(ζ_1,…,ζ_{k-1},ξ_k,ζ_{k+1},…,ζ_n)` is in `𝔗'_n`; to see this, apply a
complex Lorentz transformation `A ∈ L+(C)` that maps the point into `𝔗^n`. It
is no loss to assume `ξ_k = (0, ξ', 0, …, 0)` with `ξ' > 0`. Then the 2×2
block

```
( cos φ    i sin φ )
( i sin φ   cos φ )
```

with `sin φ > 0` small enough leaves all `ζ_l` `(l ≠ k)` inside `𝔗` while
moving `ξ_k` into `𝔗`. So:

```
(8)  W_n(ζ_1,…,ζ_{k-1}, ξ_k, ζ_{k+1},…,ζ_n)
   = W_n(ζ_1,…,ζ_{k-1}+ξ_k, -ξ_k, ζ_{k+1}+ξ_k,…,ζ_n)
     for ζ_l ∈ 𝔗 (l ≠ k), ξ_k real with (ξ_k, ξ_k) < 0.
```

Both sides converge as `Im ζ_l → 0` to their respective Wightman
distributions, and we find

```
(9)  W_n(ξ_1,…,ξ_{k-1}, ξ_k, ξ_{k+1},…,ξ_n)
   = W_n(ξ_1,…,ξ_{k-1}+ξ_k, -ξ_k, ξ_{k+1}+ξ_k,…,ξ_n)
     valid under (ξ_k, ξ_k) < 0.
```

In terms of `𝔚_{n+1}` this reads

```
(10)  𝔚_{n+1}(x_0,…,x_{k-1},x_k,…,x_n)
    = 𝔚_{n+1}(x_0,…,x_k,x_{k-1},…,x_n)
      valid whenever (x_k - x_{k-1})² < 0.    QED.
```

**Remarks.** (a) Locality is equivalent (in presence of the other axioms) to
total symmetry of `𝔚_n` in `𝔗'_n`. Any condition forcing this symmetry forces
full locality. (b) The only property of `W_n` beyond regularity in `𝔗^n` and
`L↑_+`-invariance used here is the existence of the boundary distribution
`W_n(ξ_1,…,ξ_n)` if `ζ_l → ξ_l` in `𝔗`.

### Theorem (strong locality from weak commutation)

```
Theorem.  Given that A(x) satisfies all Wightman axioms except locality and
also satisfies
                       [A(x), A(y)] = 0
for x in an arbitrary neighborhood of 0 and y in an arbitrary neighborhood
of a spacelike vector, then it satisfies the full locality axiom.
```

**Proof sketch.** Choose a real regularity point `(r_1^l,…,r_n^l)` with
`r_k^l = (k-l)·a` for some spacelike `a`. By assumption, the equation

```
(11)  𝔚_n(r_1^l,…,r_l^l, r_{l+1}^l,…,r_n^l)
    = 𝔚_n(r_1^l,…,r_{l+1}^l, r_l^l,…,r_n^l)
```

holds, and similarly in a neighborhood of this point. By analytic continuation,

```
(12)  𝔚_n(z_1,…,z_l, z_{l+1},…,z_n) = 𝔚_n(z_1,…,z_{l+1}, z_l,…,z_n).
```

Since `l` was arbitrary we obtain total symmetry of `𝔚_n`, hence full
locality. QED.

This is the cleanest paper-level statement of how locality plus
spacelike-separated commutativity in *any* neighborhood gives full locality
through analytic continuation.

## Appendix I (Normal forms of Lorentz transformations — substrate)

(Book p. 143-149; PDF p. 79-82.)

Used in Step 8 of the BHW connectivity proof and in §5 of the locality
application. Lean substrate already exists in
`ComplexLieGroups/JostPoints.lean` and `Complexification.lean`.

Key blocks:
- `K(φ)` = `((cos φ, -sin φ), (sin φ, cos φ))` — spatial rotation
- `L(χ)` = `((Ch χ, Sh χ), (Sh χ, Ch χ))` — boost in time-space block
- `M(τ)` = `((1+τ²/2, -τ²/2, -τ), (τ²/2, 1-τ²/2, -τ), (-τ, τ, 1))` — null rotation

Complex analytic continuations: `K(iχ)`, `L(iφ)`, `M(iτ)`.

The normal-form theorem (Jost III on p. 77) classifies every `A ∈ L+(C)` up
to equivalence under `L↑_+` as one of these block-diagonal patterns.

## Appendix II (Uniqueness of W(ζ) in ⋃_π 𝔗'_π — Ruelle, KEY)

(Book p. 150-152; PDF p. 83. Content credited by Jost to D. Ruelle, private
communication.)

This is the technical content that the OS-45 local Hdiff producer needs.

### Theorem (Ruelle, Jost Appendix II)

Let `π` be a permutation of `z_0, z_1, …, z_n`, with induced linear
transformation on the difference variables `ζ_k = z_k - z_{k-1}`. Let
`W(ζ_1,…,ζ_n)` be `L+(C)`-invariant, single-valued and analytic in `𝔗'_n`,
and let `W_π(ζ_1,…,ζ_n)` be `L+(C)`-invariant, analytic and single-valued in
`π𝔗'_n`. Further, assume the existence of boundary value distributions
`W(ρ_1,…,ρ_n)` and `W_π(ρ_1,…,ρ_n)`. If

```
(1)  W(ρ_1,…,ρ_n) = W_π(ρ_1,…,ρ_n)
```

as distributions on the real points, then `W_π` is a unique analytic
continuation of `W` into the domain `𝔗'_n ∪ π𝔗'_n`.

### Proof (6 steps)

Use the notation `𝔗' = 𝔗'_n`, `𝔗 = 𝔗^n`, `𝔗_π = π·𝔗`.

**Step 1.** Rule out trivial cases: `π = identity` and `π = total inversion`.

**Step 2.** From equation (1), `W_π` is `L+(C)`-invariant, so for any
`A ∈ L+(C)`, `W_π(Aρ) = W_π(ρ)`. Therefore

```
(1')  W(Aρ) = W_π(Bρ)   for A, B ∈ L+(C).
```

**Step 3.** Starting from (1) and (1') we must show

```
W(ζ) = W_π(ζ)   for ζ ∈ 𝔇 := 𝔗' ∩ 𝔗'_π.
```

Now `ζ ∈ 𝔇` ⟹ there exist `A, B ∈ L+(C)` with `ζ ∈ A𝔗 ∩ B𝔗_π`. By a
continuity argument we rule out the exceptional classes for `A` and `B`. The
intersection `𝔇_0 := A𝔗 ∩ B𝔗_π` is convex (intersection of two convex sets)
and therefore connected. It suffices to prove `W ≡ W_π` in the neighborhood
of **one** point of `𝔇_0`.

**Step 4.** Additional reduction: if `W(ζ) ≡ W_{A_0}(ζ)` for *some* Lorentz
transformation `A_0`, then they coincide also if `A` varies in a suitable
neighborhood of `A_0`. (Same identity-theorem-on-`A` argument as in BHW.)

**Step 5 (key — pinpointing the real boundary point).** Keep `ζ` fixed in
`𝔗 ∩ A_0 𝔗_π` and look at `W_A(ζ)` as a function of `A`. Introduce local
coordinates by `A = A_0 B`, with `B` parameterized in Cayley coordinates. If
`B` varies in a sufficiently small connected neighborhood of the identity in
`L+(C)`, `ζ ∈ 𝔗 ∩ A𝔗_π` for all `A` in this neighborhood, and analyticity in
the Cayley parameters of `B` holds.

For `B` real (`B ∈ L↑_+`), `W_A(ζ) = W_{A_0}(ζ)` by `L↑_+`-invariance. By the
identity theorem (`[BM 1*]`) this analytic-in-`B` function is also constant
for complex parameters. Hence `W_A(ζ) = W_{A_0}(ζ)` for all `B` in our
neighborhood, and in particular at `B = identity`. So `W ≡ W_π` in a complex
neighborhood of any chosen `ζ ∈ 𝔇_0`.

To verify the hypothesis "the chosen base case exists": consider `B = identity`
and `A` ranging over a normal form `N`. The only nontrivial case is

```
N = block-diag(L(iφ), K(iχ_1), …, K(iχ_r), [1])
```

with `φ ≠ 0, π`. Then `sin φ ≠ 0`, and `N` applied to the point

```
ζ_1 = ζ_2 = ⋯ = ζ_n = (i, -ctg φ, 0, …, 0)
```

gives

```
ρ_1 = ρ_2 = ⋯ = ρ_n = (0, -(sin φ)^{-1}, 0, …, 0)
```

which is a *real* point of `𝔗'`. So `Nρ` is a real boundary point of `𝔗'`,
where both `W` and `W_π` are regular and coincide by (1). Therefore `W ≡ W_π`
in a complex neighborhood of this `Nρ`, which lies in `N𝔗 ∩ 𝔗_π`.

**Step 6 (general case).** For general `A, B` with `A𝔗 ∩ B𝔗_π ≠ ∅`, write
`B^{-1}A = Λ_1 N Λ_2` with `Λ_i ∈ L↑_+`, `N` a normal form. Then
`B^{-1}·A𝔗 = Λ_1 N Λ_2 𝔗 = Λ_1 N 𝔗`, so the intersection contains a real
boundary point `Bρ` with `ρ` chosen as in Step 5. According to (1'),
`W = W_π` in a complex neighborhood of `Bρ`. QED.

### Application

As a direct consequence: in a *local* theory, `𝔚(z_0,z_1,…,z_n)` extends
uniquely into the permutation-extended tube `𝔖^p_n`, where the complete
symmetry in `(z_0,z_1,…,z_n)` enters through the real regularity (Jost)
points `(r_0, r_1,…,r_n)`.

For `n ≥ 2`, `𝔖^p_n` is **not** a domain of holomorphy (Jost), but the
envelope of holomorphy of `𝔖^p_n` may no longer be schlicht. This
complication does not arise for `n = 2`.

### Generalization

The boundary-value assumption can be weakened *if dimension > 2*. One only
needs `W = W_π` on the **real points** of `𝔗' ∩ 𝔗'_π`. Then equation (1)
follows by analytic continuation. This requires two properties (left to the
reader by Jost):

(a) The intersection `𝔗' ∩ 𝔗'_π` contains *real* points, if `dim 𝔅 > 2`,
hence a real neighborhood.

(b) The set of real points in `𝔗'` is connected, again if `dim 𝔅 > 2`.

This generalization gives a **purely real boundary-overlap criterion** for
the uniqueness: if the two analytic continuations agree on the real points of
the overlap, they agree everywhere.
